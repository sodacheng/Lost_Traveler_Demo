## 文件夹结构



- Assets
  - ArtRes - 美术资产
  - Game - 游戏文件夹
    - Scenes - 场景
    - Scripts - 脚本
    - Prefabs - 预制体
  - Resources - 资源文件夹, 加载方法已经重封装
  - Scripts - 框架类脚本
  - ThirdParty - 插件文件夹
    - CorgiEngine - 柯基引擎文件夹
    - Other Plugin - 其他插件
